# Aurorion Landing Project

## Overview

The Aurorion Landing project is a promotional website for our AI-powered historical guide application. It features an early access subscription system called "Pioneer Circle" that allows users to sign up for exclusive early access to the full Aurorion experience.

## Architecture Decisions

### Why Astro?
- **Performance**: Superior Core Web Vitals scores
- **DX**: Excellent developer experience with TypeScript
- **Flexibility**: Islands architecture for selective hydration
- **Cloudflare Integration**: First-class Workers support

### Why Cloudflare Platform?
- **Edge Performance**: Global distribution and low latency
- **Cost Efficiency**: Generous free tiers for MVP
- **Integrated Stack**: Workers, Pages, and D1 work seamlessly together
- **Scalability**: Built to handle traffic spikes

### Why D1 Database?
- **Simplicity**: SQLite compatibility with zero configuration
- **Performance**: Edge replication for global access
- **Cost**: No cold starts or connection limits
- **Integration**: Native Workers binding
